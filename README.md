# Domino
Se doreste realizarea unei aplicatii Web care sa implementeze jocul de domino. La cerere, se va putea simula desfasurarea unui joc de domino (i.e., reluarea unei partide) între perechi de utilizatori, calculându-se scorul obtinut de fiecare oponent la fiecare etapa a jocului. Starea unui joc aflat în desfasurare se va pastra si dupa închiderea navigatorului Web cu posibilitatea restaurarii ulterioare. Scorurile vor putea fi salvate local si partajate pe o retea sociala. Redarea partidelor în format vectorial via SVG si/sau 3D (recurgând la WebGL) va conduce la acordarea unui bonus.
